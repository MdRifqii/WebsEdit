# Bio

Sejak tahun 2020, saya telah memasuki dunia programming dan menemukan minat yang mendalam di dalamnya. Saya berasal dari Bengkulu, Indonesia, di mana saya menjalani kehidupan sehari-hari. Kesukaan saya tidak hanya terbatas pada coding, namun juga melibatkan anime, yang menjadi sumber inspirasi dan hiburan bagi saya. Saya menemukan kegembiraan dalam merakit kode dan mengembangkan solusi kreatif untuk tantangan pemrograman. Bahasa pemrograman seperti PHP dan JavaScript adalah keahlian saya, dengan kecenderungan lebih ke arah pengembangan backend.
